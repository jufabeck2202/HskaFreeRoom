# HskaFreeRoom

> get current free rooms in hska

## Docker Image Build
```
docker build  -t jufabeck2202/hskafreeroom .
```

## Docker Image Usage
```
docker run jufabeck2202/hskafreeroom
```

## Install

```
$ npm install -g  hskafreeroom
```

## Usage
```js
freeRoom
```
gives you the current free rooms.
